Welcome to MazeX!!
Rules

1. Select the difficulty level from the set of radio buttons available on top.
2. Press the "Start Game" button.
3. Use the navigation keys to pass through the grass to reach the checkered flag(goal)
   You can move only through the grass area.
4. Your objective will be to collect as much coins as possible within the time limits.
5. You lose a life if you touch the bomb. There are minimum lives available so make
   use of them effectively.
6. Some levels provide an option to gain a life(heart). Collect hearts to earn more lives.
7. Each level has one or more secret area(s) which seem to be impenetrable but are
   actually penetrable. Find thetreasure and it is all yours!
8. Last but not the least, enjoy the game
